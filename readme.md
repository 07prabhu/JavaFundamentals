DATA TYPES

****whole number/integers****

1)byte  1byte(8bits) -2^n-1 to 2^n-1 -1  | -128 to127
2)short 2bytes                              -32768 to 326767
3)int   4bytes
4)long  8bytes

default data type of any whole number is integer


****real numbers****
1)float   4bytes  
2)double  8bytes

****characters****

1) char 2bytes  (ASCII VALUES) 128(A-Z,a-z,1-9, and special characters)
2) boolean     - true/false
